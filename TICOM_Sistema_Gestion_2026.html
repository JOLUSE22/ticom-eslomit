☁️Sincronización Firebase — Tiempo Real entre PCs
¿Qué es esto? Con Firebase Realtime Database cualquier cambio que hagas se sube automáticamente a la nube y todas las PCs que tengan el sistema abierto lo ven en tiempo real — sin servidor propio.

Pasos para activarlo:
1. Creá un proyecto gratuito en console.firebase.google.com
2. Activá Realtime Database → modo prueba (o reglas de lectura/escritura autenticada)
3. Copiá los datos de configuración (SDK config) y pegálos en los campos de abajo
4. Guardá → recargá la página → el badge "Firebase" en la barra cambia a verde ✅
Configuración Firebase (editar en el archivo HTML)
Buscá en el archivo HTML la línea const FB_CONFIG = {
Completá: apiKey, databaseURL, projectId, appId
La databaseURL es la más importante — ejemplo:
https://ticom-eslomit-default-rtdb.firebaseio.com
